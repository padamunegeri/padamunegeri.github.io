3. Jelaskan tentang perintah select case berkondisi​


Jawaban:

Select Case adalah sebuah pernyataan bersyarat. If dan Select Case adalah merupakan suatu perintah untuk mengambil suatu keputusan terhadap sebuah atau beberapa kondisi. Kondisi adalah ungkapan untuk mengevaluasi sebuah pernyataan apakah bernilai benar atau salah.

Select Case biasa dipakai untuk alternatif Bentuk If..Then..Else, karena kodenya lebih mudah dibaca. Perintah SELECT CASE adalah salah satu syntax yang digunakan dalam bahasa pemrograman Visual Basic for Applications (VBA). Perintah atau syntax ini sebenarnya memiliki peran yang hampir sama dengan perintah If-Then, namun dalam beberapa kasus sedikit berbeda atau kadang tergantung selera dari programmer lebih memilih menggunakan Select Case atau perintah If-Then. Dengan kata lain SELECT CASE adalah salah satu bentuk perintah kondisi dalam Visual Basic selain IF THEN. Dengan konsep memberikan kemungkinan nilai variabel yang ingin dicari.

Intruksi atau program CASE ini marupakan alternatif dari pernyataan IF untuk masalah dengan pilihan ganda. Semua masalah yang bisa dikerjakan oleh CASE pasti bisa ditangani oleh IF, tetapi tidak sebaliknya.Keuntungan yang lain memakai intruksi CASE….OF adalah lebih singkat dan cepat, sehingga memudahkan dalam pengerjaan suatu masalah dengan pilihan ganda.

Select Case <Sebuah Variable>

  Case <Satu atau lebih kondisi>

     <Perintah yang dilakukan jika memenuhi kondisi>

  <Case Lainnya> <Kondisi yang lain>

  <Case Else>

     <Perintah yang dilakukan jika tidak memenuhi kondisi-kondisi sebelumnya>

39. 1. Tool select digunakan untuk perintah … 2. Perintah yang digunakan untuk memutar gambar adalah … semoga jawab


Jawaban:

1.Beberapa Perintah Tambahan dalam Tool Seleksi Photoshop – Dalam aplikasi Photoshop terdapat banyak perintah-perintah tambahan untuk mempermudah dalam pekerjaan pengeditan pada Photoshop. Menggunakan Photoshop lebih mengandalkan tool, dan terdapat juga tool seleksi.Tool Seleksi adalah bagian yang sangat penting dalam memanipulasi gambar, tanpa adanya Tool Seleksi ini maka sulit bagi para user untuk melakukan Editing ataupun untuk memanipulasi objek dan lain sebagainya. Selain itu juga, Tool Seleksi berfungsi sebagai pembatas area gambar yang mana area gambar tersebut di batasi ketika ingin melakukan action seperti memberi warna di area yang di seleksi sehingga area yang tidak di seleksi tidak ikut terkena warna

Beberapa Perintah Tambahan dalam Tool Seleksi Photoshop

1. All

Perintah All digunakan untuk memilih (seleksi) semua yang ada pada lapisan layer aktif dan diratakan pada pinggir gambar.

2. Deselect

Perintah Deselect untuk menghilangkan seleksi yang sedang aktif.

3. Reselect.

Perintah Reselect digunakan untuk dapat memilih kembali seleksi yang terakhir dibuat pada dokumen anda.

4. Inverse.

Inverse digunakan untuk membalikkan seleksi dari yang tidak kita seleksi menjadi yang sebenarnya yang ingin kita seleksi.

5. Grow.

Perintah Grow membuat seleksi pada pixel yang berdekatan dengan batas toleransi tertentu. Untuk merubah jangkauan Grow dapat di atur pada Toleransi di Magic Wand tool.

6. Similar.

Perintah ini juga memilih pixel berdasarkan toleransi Magic Wand Tools

7. Transform Selection.

Transform Selection berfungsi untuk memodifikasi seleksi yang dibuat seperti menggeser, merubah ukuran, atau memutarnya.

Sedangkan untuk perintah yang terdapat pada submenu Modify adalah

1. Border.

Perintah Border digunakan untuk membuat frame baru setelah menentukan seleksi.

2. Smooth.

Perintah Smooth untuk membuat seleksi lebih halus dengan menambahkan nilai piksel yang lebih besar pada daerah yang bergerigi.

3. Expand.

Perintah expand berfungsi untuk memperluas daerah seleksi berdasarkan nilai piksel yang diberikan.

4. Contract.

Perintah Contract untuk menentukan jumlah piksel sesuai yang diinginkan untuk membuatnya lebih kecil.

5. Feather.

Fungsi Feather berfungsi untuk mengaburkan tepi seleksi.

2.Memutar tabel atauGrafik SmartArt

Salin tabel atau Grafik SmartArt, tempelkan sebagai gambar, lalu putar gambar tersebut.
