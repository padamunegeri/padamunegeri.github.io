5. pengertian dan fungsi disk compression?


Pengertian dan fungsi disk compression adalah salah satu perangkat lunak utilitas yang bertugas dalam mengcompress/mengecilkan/menekan file sebelum dilakukan penyimpanan oleh sistem operasi

Pembahasan

Disk Compression adalah perangkat lunak utilitas yang terletak sebagai jembatan antara sistem operasi dengan disk drive. Disk compression sendiri dapat bertindak untuk melakukan compress atau decompress. Fungsi ini sendiri bergantung pada kondisi berikut :

Jika suatu file akan disimpan di sistem operasi, maka disk compression akan melakukan compress pada file tersebut sebelum disimpanJika suatu file akan dibuka kembali oleh sistem operasi maka disk compression ini akan kembali melakukan decompress pada file tersebut

Salah satu keuntungan adanya disk compression adalah dapat mengefektifkan kapasitas penyimpanan dalam disk. Disk compression umumnya tersedia sebagai bawaan dari Windows yang bernama DriveSpace walau tidak menutup kemungkinan juga jika pengguna ingin menggunakan perangkat lunak selain built-in Windows.

Berkaitan dengan perangkat lunak utilitas, perangkat lunak jenis ini merupakan bagian dari perangkat lunak sistem.

19. cache browser adalah


Cache browser, kadang-kadang juga disebut sebagai folder file sementara Internet, berisi file dari situs web yang Anda kunjungi. Semua web browser modern menjaga file cache untuk satu alasan penting, tujuan nya adalah untuk menampilkan halaman web lebih cepat di waktu berikutnya apabila Anda membuka lagi situs-situs web tersebut. 22. Tuliskan pengertian keyboard, motherboard, hard disk drive dan printer


Jawaban:

- pengertian keyboard

Keyboard adalah sebuah perangkat keras (hardware) pada komputer yang berfungsi sebagai alat untuk input data yang berupa huruf, angka dan simbol. Selain sebagai alat untuk men-inputkan data keyboard juga memiliki berbagai macam fungsi tombol dengan kombinasi yang khusus.

- motherboard

adalah papan sirkuit cetak utama di komputer tujuan umum dan sistem lain yang dapat diperluas. Ini memegang dan memungkinkan komunikasi antara banyak komponen elektronik penting dari suatu sistem, seperti unit pemrosesan sentral dan memori, dan menyediakan konektor untuk periferal lain.

- hard disk drive

Hardisk adalah perangkat keras yang digunakan untuk menyimpan data pada komputer. Semua data yang ada dalam komputer baik itu dokumen, gambar, data sistem, OS, aplikasi akan tersimpan dalam perangkat ini.

- printer

printer adalah peranti yang menampilkan data dalam bentuk cetakan, baik berupa teks maupun gambar/grafik, di atas kertas. Pencetak biasanya terbagi atas beberapa bagian, yaitu penggetil sebagai alat mengambil kertas dari baki. Baki ialah tempat menaruh kertas.
