= panjang x lebar
= (40-12+12) x 10
= 16cm x 10cm = 160 cm persegi
4. Luas bangun 2
= panjang x lebar
= 40cm x 19cm = 760 cm persegi

= luas bangun 1 + luas bangun 2
= 160 + 760
= 920 cm persegi
