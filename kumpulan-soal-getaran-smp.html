9. Soal No. 3Periode suatu gelaran adalah 112 detik Tentukana) frekuensi getaranby jumlah getaran dalam 5 menit​


a. frekuensi getaran adalah [tex]\boxed{\bold{0,0089 \: Hz}}[/tex]

b. banyak getaran dalam 5 menit adalah [tex]\boxed{\bold{2,678 \: kali}}[/tex]

Pembahasan

Hi! soal tersebut adalah membahas mengenai getaran dan gelombang. Singkatnya, getaran adalah pergerakan berbolak balik di sekitar keseimbangan. Biasanya getaran di ilustrasikan dengan bandul atau gerakan pegas.

Rumus menentukan frekuensi :

[tex]\boxed{\bold{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\bold{f}[/tex] = frekuensi.

[tex]\bold{n}[/tex] = jumlah getaran dalam waktu t sekon.

Berarti :

[tex]\boxed{\bold{f = \frac{1}{T}}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\bold{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\bold{T}[/tex] = periode.

[tex]\bold{n}[/tex] = banyak getaran.

[tex]\bold{t}[/tex] = waktu.

Berarti :

[tex]\boxed{\bold{T= \frac{1}{f}}}[/tex]

Langkah Penyelesaian

Diketahui :

Periode = 112 sekon.

Ditanya :

a. Frekuensi getaran adalah?

b. Banyak getaran dalam 5 menit adalah?

Dijawab :

Mencari frekuensi :

[tex]\bold{f = \frac{1}{T}}[/tex]

[tex]\bold{f = \frac{1}{112}}[/tex]

[tex]\bold{f = 0,0089 \: Hz}[/tex]

Mencari banyak getaran dalam 5 menit :

5 menit = 300 detik.

[tex]\bold{n = \frac{t}{T}}[/tex]

[tex]\bold{n = \frac{300}{112}}[/tex]

[tex]\bold{n = 2,678 \: kali}[/tex]

Kesimpulan :

a. frekuensi getaran adalah [tex]\boxed{\bold{0,0089 \: Hz}}[/tex]

b. banyak getaran dalam 5 menit adalah [tex]\boxed{\bold{2,678 \: kali}}[/tex]

20. Apabila sebuah benda bergetar sebanyak 600 getaran setiap menit, berapakah periode getaranya?


Apabila sebuah benda bergetar sebanyak 600 getaran setiap menit, maka periode getaranya adalah 0,1 sekon

PEMBAHASAN

Gelombang sinusoidal adalah gelombang yang membentuk fungsi sinus. Beberapa rumus dasar gelombang yang perlu di ingat adalah sebagai berikut ini :

f = 1 / T

v = λ . f

Δx / λ = ΔΘ / 2π

Δt / T = ΔΘ / 2π

dimana :

f = frekuensi gelombang ( Hz )

T = periode gelombang ( s )

v = cepat rambat gelombang ( m/s )

λ = panjang gelombang ( m )

Δx = jarak pisah antar dua titik pada gelombang ( m )

Δt = selang waktu ( s )

ΔΘ = beda fase ( radian )

Bentuk persaman gelombang adalah :

y = A sin ( ωt - kx )

dimana :

y = simpangan gelombang ( m )

A = amplitudo gelombang ( m )

ω = frekuensi sudut ( rad/s )

t = waktu ( s )

k = bilangan gelombang ( 1/m ) = 2π/λ

x = jarak titik dari sumber ( m )

Okay mari kita gunakan rumus ini untuk menyelesaikan soalnya.

Diketahui :

n = 600

t = 1 menit = 60 detik

Ditanyakan :

T = ?

Penyelesaian :

T = t / n

T = 60 / 600

T = 0,1 sekon

33. Jika banyak getaran yang dihasilkan setiap detiknya adalah 20 getaran, maka periode gataran terebut adalah​


Periode getaran tersebut adalah [tex]\boxed{\sf{0,05 \: s}}[/tex]

Pembahasan

Hi! soal tersebut adalah membahas mengenai getaran dan gelombang. Singkatnya, getaran adalah pergerakan berbolak balik di sekitar keseimbangan. Biasanya getaran di ilustrasikan dengan bandul atau gerakan pegas.

Rumus menentukan frekuensi :

[tex]\boxed{\sf{f = \frac{n}{t}}}[/tex]

Dimana :

[tex]\sf{f}[/tex] = frekuensi. [tex]\sf{n}[/tex] = jumlah getaran dalam waktu t sekon.[tex]\sf{t}[/tex] = waktu.

Berarti :

[tex]\sf{f = \frac{1}{T}}[/tex]

Rumus menentukan periode :

[tex]\boxed{\sf{T = \frac{t}{n}}}[/tex]

Dimana :

[tex]\sf{T}[/tex] = periode. [tex]\sf{n}[/tex] = banyak getaran. [tex]\sf{t}[/tex] = waktu.

Berarti :

[tex]\sf{T= \frac{1}{f}}[/tex]

Langkah Penyelesaian

Diketahui :

Banyak getaran = 20 getaran.

Waktu = 1 s.

Ditanya :

Periode getaran tersebut adalah ?

Dijawab :

[tex]\sf{T = \frac{t}{n}}[/tex]

[tex]\sf{T = \frac{1}{20}}[/tex]

[tex]\sf{T = 0,05 \: s}[/tex]

Kesimpulan : Jadi periode getaran tersebut adalah [tex]\boxed{\sf{0,05 \: s}}[/tex]

39. Jika banyaknya getaran yang dihasilkan setiap detiknya adalah 2getaran maka periode getaran tersebut adalah


JAWABAN

Periode getaran tersebut adalah 0,5 sekon.

PEMBAHASAN

[tex] \boxed{ \text{Menentukan periode getaran}}[/tex]

Periode getaran merupakan waktu yang diperlukan untuk melakukan satu kali getaran.

Untuk menentukan periode getaran, kita bisa menggunakan rumus berikut.

[tex] \boxed{T = \frac{t}{n}}[/tex]

Keterangan :

T = periode getaran (sekon)

t = waktu getaran (sekon)

n = banyak getaran

PENYELESAIAN

Diketahui :

t = 1 sekon

n = 2 getaran

Ditanya :

T = ?

Jawab :

T = t/n

T = 1/2

T = 0,5 sekon

~ Kesimpulan ~

Jadi, periode getaran tersebut adalah 0,5 sekon.
