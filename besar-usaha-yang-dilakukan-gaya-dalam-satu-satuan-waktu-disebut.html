PEMBAHASAN

Mari kita ingat kembali rumus tentang Usaha , Energi Kinetik dan Energi Potensial Gravitasi sebagai berikut ini:

[tex]\boxed{W = F \cdot s}[/tex]

[tex]\boxed{Ek = \frac{1}{2} m v^2}[/tex]

[tex]\boxed{Ep = m g h}[/tex]

dimana:

W = Usaha ( J )

F = Gaya ( N )

s = Perpindahan ( m )

Ek = Energi Kinetik ( J )

Ep = Energi Potensial Gravitasi ( J )

m = massa benda ( kg )

v = kecepatan benda ( m/s )

g = percepatan gravitasi ( m/s² )

h = ketinggian benda ( m )

[tex]\texttt{ }[/tex]

Usaha adalah hasil kali gaya dengan besarnya perpindahan yang searah dengan gaya tersebut. Usaha termasuk ke dalam besaran skalar yang artinya hanya memiliki nilai ( besar ) tanpa arah. Usaha juga bisa diartikan sebagai besarnya perubahan energi pada benda.

[tex]\texttt{ }[/tex]

Daya adalah besarnya usaha yang dilakukan gaya dalam satu satuan waktu yang bisa dirumuskan sebagai berikut ini:

[tex]\boxed{P = \frac{W}{t}}[/tex]

dimana:

P = besar daya ( Watt )

W = besar usaha ( Joule )

t = selang waktu ( sekon )
