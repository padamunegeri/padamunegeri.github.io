14. jelaskan bagian" desktop windows xp


sama aja sih sma win 7, cm beda bantuk. pertama star menu, status bar, icon bar, trs tanggal.
a. Wallpaper

Wallpaper adalah gambar menempel di atas desktop. Windows menyediakan berbagai macam pilihan walpaper sehingga kita dapat dengan mudah mengubahnya sesuai dengan keinginan .

b. Ikon shortcut

shortcut adalah gambar kecil yang mewakili file,objek,folder,program aplikasi,dan drive pada komputer. ikon shortcut ini untuk memudahkan dalam mengaktifkan suatu program aplikasi. Ikon ini bersifat semipermanen sehingga jumlahnya dapat dikurangi maupun di tambah . Namun,secara default ikon shortcut dalam desktop sebagai berikut:
1) My computer, ikon ini berfungsi untuk mengelola file dan folder pada windows.
2)My document , merupakan suatu folder standart pada windows di mana semua data secara otomatis akan tersimpan pada folder ini
3)internet explorer ,ikon ini berfungsi untuk mengaktifkan software internet explorer. internet explorer adalah ikon yang digunakan untuk menjelajahi dunia maya.
4)Recycle Bin,kegunaan ikon ini adalah untuk menampung sementara file yang kita hapus.
5)Beberapa ikon program aplikasi yang di sesuaikan dengan kegunaan masing masing.

c. Taskbar

Taskbar adalah sarana yang digunakan untuk membuka program serta menjelajahi komputer.Taskbar terletak pada bagian bawah layar windows baik windows XP ataupun Windows 98.
bagian baginnya adalah:
1)start adalah tombol yang berisi menu program . jika tombol ini di klik akan tampil daftar program dan aplikasi yang di jalankan .
2)ikon program adalah ikon yang menandakan jendela sedang di buka pada saat itu .
aplikasi yang redup menandakan bahwa aplikasi sedang berjalan tetapi tidak di gunakan , sedangkan aplikasi yang terang menandakan program sedang berjalan .
3) Toolbars Quick launch berisi tombol tombol yang dapat di klik untuk melakukan atau menjalankan perintah perintah tertentu. Quick launch biasanya berisi program yang sering di gunakan

1. DEKSTOP a/ tampilan (display) pada layar komputer yang terdiri atas latar belakang (background) dan ikon-ikon yang mewakili program atau file tertentu.

2. 1. a) Start menu, yang bulet dan paling kiri itu fungsinya untuk menampung shortcut-shortcut perintah untuk mengakses dan menjalankan program, dokumen, setting dll.

b) Middle section, bagian tengah yang menampilkan program-program dan files yang sedang anda buka dan memungkinkan untuk melakukan switch secara cepat. Misalnya program aplikasi yang sedang dijalankan.
