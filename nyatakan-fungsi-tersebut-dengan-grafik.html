7. fungsi kuadrat dinyatakan oleh f(x)=X²-4X+3 sketsa grafik fungsi tersebut ditunjukkan oleh ​


Jawab:

Grafik fungsi kuadrat yang tepat ialah E

Penjelasan dengan langkah-langkah:

Untuk mengetahui sketsa grafik yang tepat dari persamaan x²-4x+3, pertama kita bisa mencari titik puncak, titik potong sumbu x, dan titik potong sumbu y

a = 1, b = -4, c = 3

Titik puncak (Xp, Yp)

Xp = -b/2a = -(-4)/2(1) = 4/2 = 2

Yp = -D/4a = - (b² - 4ac)/4a = - ((-4)²-4(1)(3))/4(1) = -(16-12)/4 = -4/4 = -1

Titik puncak (2, -1)

Titik potong sumbu x (y = 0)

x² - 4x + 3 = 0

(x - 1)(x - 3)

x = 1, x = 3

Titik potong sumbu x (1, 0) dan (3, 0)

Titik potong sumbu y (x = 0)

y = x² - 4x + 3 = 0² - 4(0) + 3 = 3

Titik potong sumbu y (0, 3)

Sesuai keterangan di atas, maka grafik fungsi kuadrat yang tepat ialah E.

Ringkasan, grafik melalui di titik ( -3 , 0 ), ( 1 , 0 ), ( 0 , 3 ) dan ( -1 , 4 )

Titik perpotongan dengan garis x

Y = -x² - 2x + 3

-x² - 2x + 3 = 0

- ( x² + 2x - 3 ) = 0

- ( x + 3 ) ( x - 1 ) = 0

Maka fungsi kuadrat memotong sumbu x pada ( -3 , 0 ) dan ( 1 , 0 )

Y = -x² - 2x + 3

Y = ax² + bx + c

Y = -x² - 2x + 3

x = -b / 2a

x = - ( - 2 ) / 2 (-1 )

x = 2 / -2

x = -1

--- Substitusikan nilai x ke fungsi kuadrat untuk mencari y ---

Y = -x² - 2x + 3

y = - ( -1 )² - 2 ( -1 ) + 3

y = -1 + 2 + 3

y = 4

Titik puncak : ( -1 , 4 )

15. ● Pernyataan yang benar untuk grafik fungsi di atas adalah....​


Pernyataan yang benar untuk grafik fungsi [tex]\displaystyle{f(x)=\frac{1}{3}x^3-2x^2-12x+27}[/tex] adalah (B) (1)  dan (3)  SAJA yang benar .

PEMBAHASAN

Turunan atau Diferensial merupakan pengukuran terhadap bagaimana fungsi berubah seiring perubahan nilai input. Salah satu fungsi dari turunan adalah menentukan nilai minimum/maksimum dari suatu fungsi. Dimana suatu fungsi akan memiliki nilai minimum/maksimum pada saat :

f'(x) = 0

dengan :

f'(x) = turunan pertama fungsi.

Dari f'(x) = 0 kita akan memperoleh titik titik stasioner, misal x = a. Untuk menentukan apakah titik tersebut menyebabkan fungsi bernilai minimum atau maksimum dapat kita gunakan uji turunan kedua.

1. Jika f''(a) > 0 maka x = a menyebabkan fungsi bernilai minimum.

2. Jika f''(a) < 0 maka x = a menyebabkan fungsi bernilai maksimum.

3. Jika f''(a) = 0 maka x = a merupakan titik belok fungsi.

Salah satu fungsi dari turunan yang lain adalah menentukan interval fungsi naik atau fungsi turun, dimana :

1. Fungsi akan naik pada saat f'(x) > 0.

2. Fungsi akan turun pada saat f'(x) < 0.

.
DIKETAHUI

[tex]\displaystyle{f(x)=\frac{1}{3}x^3-2x^2-12x+27}[/tex]

.

DITANYA

Tentukan pernyataan yang benar :

(1) Mencapai maks relatif di x = -2 dan min relatif di x = 6.

(2) Titik belok [tex]\displaystyle{\left ( 2,-\frac{7}{3} \right )}[/tex]

(3) Turun untuk -2 < x ≤ 3.

(4) Naik untuk x > 4.

.

PENYELESAIAN

> Mencari titik ekstrim fungsi.

[tex]\displaystyle{f(x)=\frac{1}{3}x^3-2x^2-12x+27}[/tex]

[tex]f'(x)=x^2-4x-12[/tex]

.

Cari titik stasioner.

[tex]f'(x)=0[/tex]

[tex]x^2-4x-12=0[/tex]

[tex](x+2)(x-6)=0[/tex]

[tex]x=-2~atau~x=6[/tex]

.

Cek uji turunan kedua.

[tex]f'(x)=x^2-4x-12[/tex]

[tex]f''(x)=2x-4[/tex]

Substitusi titik stasioner ke f''(x) :

[tex]f''(-2)=2(-2)-4=-8~(< 0)[/tex]

[tex]f''(6)=2(6)-4=8~(> 0)[/tex]

Dari uji turunan kedua dapat disimpulkan :

x = -2 menyebabkan fungsi mencapai nilai maksimum relatif.

x = 6 menyebabkan fungsi mencapai nilai minimum relatif.

Fungsi tidak memiliki titik belok.

⇒ Pernyataan (1) benar, pernyataan (2) salah.

.

> Mencari interval fungsi naik dan turun.

Interval naik :

[tex]f(x)> 0[/tex]

[tex]x^2-4x-12> 0[/tex]

[tex](x+2)(x-6)> 0[/tex]

Cek menggunakan uji titik.

[tex]++o--o++[/tex]

[tex].~~-2~~~~~6[/tex]

Karena tanda pertidaksamaan > 0, pilih daerah bertanda ++, yaitu x < -2 atau x > 6.

.

Interval turun :

[tex]f'(x)< 0[/tex]

[tex](x+2)(x-6)< 0[/tex]

Dari uji titik di atas pilih daerah bertanda --, yaitu -2 < x < 6.

Dapat disimpulkan :

Fungsi naik pada interval x < -2 atau x > 6.

Fungsi turun pada interval -2 < x < 6.

⇒ Pernyataan (3) benar, pernyataan (4) salah.

17. perhatikan grafik fungsi kuadrat f(x)=ax²+bx+c berikut. pernyataan berikut yang benar tentang grafik di atas adalah


Perhatikan grafik fungsi kuadrat f(x) = ax² + bx + c berikut. Pernyataan berikut yang benar tentang grafik di atas adalah a < 0, b < 0, c < 0 dan D < 0. Bentuk umum fungsi kuadrat adalah f(x) = ax² + bx + c, dengan a ≠ 0. Diskriman pada fungsi kuadrat berfungsi untuk menentukan apakah fungsi kuadrat tersebut memotong sumbu x atau tidak. Rumus diskriminan: D = b² – 4ac  

Jika D > 0, maka grafik fungsi kuadrat memotong sumbu x di dua titik Jika D = 0 maka grafik fungsi kuadrat memotong sumbu x di satu titik (menyinggung sumbu x) Jika D < 0 maka grafik fungsi kuadrat tidak memotong sumbu x (definit)

Bentuk grafik fungsi kuadrat adalah parabola yang

Terbuka ke atas jika a > 0 Terbuka ke bawah jika a < 0

Titik puncak fungsi kuadrat berada

disebelah kanan sumbu y jika ab < 0 disebelah kiri sumbu y jika ab > 0 pada sumbu y jika ab = 0

Fungsi memotong sumbu y  

Di atas sumbu x jika c > 0 Di bawah sumbu x jika c < 0 Pada sumbu x jika c = 0

Pembahasan

f(x) = ax² + bx + c, pada gambar diketahui:

Kurva terbuka ke bawah maka a < 0

Titik puncak fungsi kuadrat berada di sebelah kiri sumbu y maka ab > 0 . Karena a < 0 maka agar ab > 0, haruslah b < 0

Grafik memotong sumbu y di bawah sumbu x maka c < 0

Grafik tidak memotong sumbu x maka D < 0

Jadi pernyataan yang benar adalah  

C. a < 0, b < 0, c < 0 dan D < 0

Fungsi kuadrat f(x) = 4x² - 4x + 1. Maka pernyataan yang benar adalah A dan D.

Penjelasan dengan langkah-langkah

Diketahui :

f(x) = 4x² - 4x + 1

Ditanya :

Pernyataan yang benar di bawah ini adalah.....

A. grafik fungsi tidak memotong sumbu-x

B. grafik fungsi memotong sumbu-x di satu titik

C. grafik fungsi memotong sumbu-x di dua titik

D. grafik fungsi hanya memotong sumbu-y -​

Jawab :

Memeriksa pernyataan.

A. grafik fungsi tidak memotong sumbu-x

Jika y = 0 ⇒ 4x² - 4x + 1 = 0

                ⇔ (2x - 1)(2x - 1) = 0

                ⇔ 2x - 1 = 0

                ⇔ x = 1/2

Grafik fungsi memotong sumbu X di titik (1/2, 0)

Maka pernyataan A. Salah.

B. grafik fungsi memotong sumbu-x di satu titik

Pernyataan B. Benar, karena memotong sumbu X di titik (1/2, 0).

C. grafik fungsi memotong sumbu-x di dua titik

Pernyaan C. Salah

D. grafik fungsi hanya memotong sumbu-y -​

Jika x = 0 ⇒ y = 4x² - 4x + 1

                    y = 4(0)² - 4(0) + 1

                    y = 1

Grafik fungsi memotong sumbu Y di titik (0, 1).

Pernyataan D. Benar

Jadi pernyataan yang benar adalah A dan D.
