2. Apa pengertian dari ROM​


Read Only Memory atau dapat disingkat dengan ROM yang mempunyai kemampuan untuk menyimpan data secara permanen. Ketika sumber dayanya mati, ROM tidak akan kehilangan datanya, lantaran dijalankan serta ditopang oleh baterai kecil yang terpasang pada sebuah komputer. ROM ini memiliki fungsi utama untuk input dan output data, serta memegang kendali atas program perangkat lunak.  

Pembahasan

Biasanya ROM berbentuk chip serta terletak di motherboard dan papan sirkuit. Tujuannya adalah untuk mengatur input dan output dari sistem manajemen data dasar, serta membaca dan menulis ke perangkat.  

ROM ini menyediakan instruksi yang diperlukan tentang cara perangkat saling terhubung dengan komponen lain. ROM sendiri biasanya digunakan sebagai firmware. Firmware merupakan sekumpulan program yang ditanamkan ke dalam mekanisme sebuah perangkat keras komputer atau hardware. ROM umumnya dapat dijumpai di antara media penyimpanan optik, seperti CD-ROM, CD-R atau singkatan dari Compact Disc Recordable, dan CD-RW atau singkatan dari Compact Disc Re-Writeable.  

Secara sederhana, ROM dapat disebut sebagai tempat untuk menyimpan data internal yang juga bisa untuk menjalankan sistem operasi serta sistem yang lainnya.

ROM ini penyimpanannya permanen, Non-volatile atau tidak mudah 'menguap', kemudian digunakan untuk proses startup komputer, serta menulis datanya lebih lambat.

4. Pengertian cd rom, cd-r,cd+r,cd rw


CD-ROM  merupakan akronim dari “compact disc read-only memory” adalah sebuah piringan kompak dari jenis piringan optik (optical disc) yang dapat menyimpan data. Ukuran data yang dapat disimpan saat ini bisa mencapai  700MB atau 700 juta bita CD-ROM drive hanya bisa digunakan untuk membaca sebuah CD saja. Secara gari besar CD-ROM dibedakan menjadi 2 menurut tipenya yaitu : ATA/IDE  dan SCSI. Yang paling mendasari dari perbedaan tersebut adalah kecepatannya. Kalau ATA memiliki kecepatan 100-133Mbps sedangkan SCSI memiliki kecepatan kira-kira 150 Mbps. Untuk tipe SCSI biasanya ditemuka pada CR RW drive. Pada CD ROM terdapat tulisan 56X artinya kemampuan memberikan kecepatan transfer data sebesar 56 x150 Kbps. Tipe CD RW juga biasanya  dibedakan berdasarkan kemapuan membakar dan membaca. CD RW tipe 12x8x32 artinya memiliki kemampuan membakar pada CD R seccepat 12x, membakar pada CD RW secepat 8x, dan membaca CD R/CD RW/dengan kecepatan maksimal 32x.

15. jelaskan Jelaskan perbedaan rom dan ram serta jelaskan pengertian masing-masing ​


Jawaban:

Definisi :

RAM

Singkatan dari Random Access Mermory adalah sebuah hardware untuk penyimpanan sementara program & data program pada saat menjalaskan aplikasi atau program. Di karenakan sementara atau Volatile jadi jika PC/Laptop mati data pada RAM akan kosong.

ROM

Singkatan dari

Read Only Memory adalah sebuah hardware chip memory semi konduktor yang terletak pada Mobo<Mother Board>. Biasanya ini digunakan untuk menyimpan data atau frimware <BIOS/UEFI> default bawaan pabrik yang bisa di baca oleh perangkat. ROM termasuk memory non-volatile jadi data tidak akan hilang jika PC/Laptop di matikan.

Jenis-Jenis

RAM :

1. D RAM <Dynamic RAM>

     RAM yang di refresh oleh CPU <Prosesor> agar data-data didalamnya tidak hilang, kekurangannya hanya bisa menyimpan sedikit data

2. SD RAM < Sychronous Dynamic RAM>

   teknologi lanjutan dari D RAM, SD RAM sudah di sinkronisasikan oleh clock system dan lebih cepat dari D RAM

3. S RAM <Static RAM>

   RAM ini berbeda dari teknologi terdahulunya, RAM jenis ini dapat menyimpan data tanpa harus di refresh dari CPU <Prosesor> dan bekerja lebih cepat dari D RAM & SD RAM

4. DDR RAM < Double Rate RAM>

   RAM ini yang sering kita gunakan di PC/Laptop kita, Teknologi ini lebih maju dari yang lainnya dan terus memunculkan generasi terbaru seperti DDR 2, DDR 3, DDR4 <yang kita gunakan>, DDR 5 <Rencana dan sudah ada beberapa mobo yg ada dengan teknologi ini>

5. RDR RAM <Rambus Dynamic Access Memory>

  RAM ini digunakan pada system Pentium 4 dan bekerja lebih cepat dari SD RAM dan Lebih mahal.

6. EDO RAM < Extended Data Out RAM>

  RAM untuk OS Pentium, dan mampu menyimpan sekaligus mengambil data bersamaan

ROM

1. Stock ROM

   Frameware bawaan dari pabrik dan bersifat default, standar dan belum dimodif

2. Custom ROM

   Frameware yang sudah dimodifikasi

3. Manufacturer < Carrier Stock ROM>

   Frameware yang di modif oleh pabrik agar bisa di costum pada aplikasi

4. Mask ROM

   Data ditulis saat chip di buat

5. PROM <Programmable Read Only Memory>

   Data ditulis setelah chip di buat

6. EPROM <Eraseable Programmable Read Only Memory>

   bisa di hapus sengan sinar UV yang sangat kuat

7. EEPROM <Electrically Erasable Programmable Read Only Memory>

  data bisa dihapus dengan teknolofi elektrik seperti emisi elektron

Fungsi

RAM

1. Membaca Data

     membaca data dari HDD/SDD/SSHD  pada saat dibuka

2. Penyimpan Sementara

   data-data program/aplikasi pada saat berjalan

ROM

Penyimpanan frimware <BIOS/UEFI> pada Mobo<mother board>

Cara Kerja

RAM

Bekerja pada saat program/ aplikasi dalam PC/Laptop di jalan kan

untuk Windows  untuk melihat apa saja aplikasi/program yg sedang di jalankan oleh user dan system dapat di lihat di task manager

ROM

Hanya dapat di baca oleh user. Karena data-data sofware secara default adalah manufaktur dari Mobo <Mother Board> tersebut.
