7. tuliskan pengertian dari query engine dan berikan contohnya​


Jawaban:

Pengertian Query Engine adalah sebuah sistem atau alat yang digunakan untuk mencari informasi dalam database atau sumber data yang besar. Query engine mengolah permintaan pencarian dari pengguna dan mengambil informasi yang sesuai dari database. Hasil pencarian dapat ditampilkan dalam bentuk tabel, grafik, atau laporan.

Contoh dari Query Engine adalah search engine seperti Google, Bing, atau Yahoo! Search. Pengguna dapat memasukkan kata kunci atau frasa tertentu pada mesin pencari untuk menemukan informasi yang diinginkan. Mesin pencari akan menggunakan Query Engine untuk mengolah permintaan pencarian dan menemukan informasi yang sesuai dalam database mereka. Hasil pencarian akan ditampilkan dalam bentuk hasil web yang relevan dengan kata kunci atau frasa yang dimasukkan.

Buatlah query SQL untuk menampilkan Data Buku yang dikarang oleh Willy atau penerbitnya adalah Gagas Media​


Jawaban:

Gunakan query where clause:

1. SELECT * FROM buku_teknik_informatika WHERE tahun_terbit=2015 AND penerbit='Gramedia'

2. SELECT * FROM buku_teknik_informatika WHERE pengarang='Willy' OR penerbit='Gagas Media'

Catatan :

query AND adalah brrti kedua kondisi harus terpenuhi. ( lihat pertanyaan 1, disitu tertulis 'dan' ).

pada pertanyaan 1 dalam prosesnya akan mencari dimana kondisi tahun_terbit harus 2015 dan penerbit nya Gremedia. Selama ke dua kondisi tersebut tidak dipenuhi maka tidak diperoleh output.

query OR adalah berrti salah satu kondisi harus trpenuhi ( lihat pertanyaan 2 disitu tertulis 'atau' )

pada pertanyaan 2 dalam prosesnya akan mencari dimana kondisi pengarang nya Willy atau penerbitnya Gagas Media. Ini berarti jika dalam proses pncarian ditemukan kecocokan salah satu dari 2 kondisi tersbut, maka data akan ditampilkan.

39. apa yang dimaksud dengan query?


Query adalah suatu bentuk pemanggilan data yang ada pada database. Pemanggilan data menggunakan query harus memenuhi format dan aturan yang sesuai dengan ketentuan Structure Query Language/SQL agar database dapat membaca pemanggilan secara tepat. Penggunaan query dapat digunakan untuk melakukan menambahkan data, mengupdate data dan melakukan penghapusan data.

Pembahasan

Query merupakan suatu hal yang identik dengan database karena keduanya memiliki hubungan. Tanpa query, data yang tersimpan dalam database tidak dapat dilakukan manipulasi terhadap isi didalamnya karena diibaratkan pengguna melakukan requesting terhadap database untuk melakukan tindakan terhadap data didalamnya. Contoh query adalah penggunaan SQL statement seperti SELECT [column, ...] FROM [table name]. Untuk mengakses data dalam database, ada cara lain selain menggunakan query, namun, query tetap dianggap solusi paling efektif dalam melakukan pemanggilan data di database.
