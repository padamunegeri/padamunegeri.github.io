15. Jelaskan pengertian, fungsi, kegunaan, tab menu (ribbon) dari Ms. Excel


Ribbon pada Excel terdiri dari tab menu, grup perintah (command group) dan tombol perintah (command button). Tab Menu pada Microsoft Excel 2016 terdiri dari menu File, Home, Page Layout, Formulas, Data, Review, View, Developer, Load Test, Power Pivot dan Team. Fitur Developer, Load Test, Power Pivot dan Team secara default disembunyikan pada Microsoft Excel

Masing-masing menu terdiri dari command group tempat tombol perintah dikelompokan. Masing-masing tombol perintah memberikan beragam kegunaan atau Fungsi Ribbon pada Microsoft Excel. Dapat disimpulkan Pengertian Ribbon pada Excel adalah kumpulan perintah yang terdiri dari beberapa tab menu, yang mana setiap tab menu mengelompokan tombol perintah dalam grup tertentu.

Ribbon Microsoft Excel terdiri dari dari tab menu yang dilengkapi dengan grup perintah, berikut penjelasannya

Menu File

Menu file merupakan menu utama pada Microsft Excel 2016. Sejak dirilisnya Microsoft Excel 2010, Office Button mulai digantikan dengan Menu File pada Ribbon Microsoft Excel. Menu File berfungsi untuk membuat, menyimpan, membuka serta manajemen file dan aplikasi pada Microsoft Excel.

Clipboard adalah grup perintah yang digunakan untuk melakukan cut, copy, paste pada Microsoft Excel. Cut adalah perintah untuk memindahkan data, copy perintah untuk menggandakan data dan paste adalah perintah untuk mengekseskusi cut/paste di sel, worksheet maupun workbook yang berbeda dengan data aslinya.

Font adalah grup perintah yang berfungsi untuk managemen font pada Microsoft Excel. Anda dapat membuka dialog font dengan menekan ctrl+shift+f. Berikut dialog font pada Microsoft Excel

1. 8 macam menu / tab:

1. Tab menu file

Dalam tab menu file berisi ikon-ikon sebagai berikut :

Save : untuk menyimpan dokumen yang telah disimpan setelah diubah.Save As : untuk meyimpan dokumen baru atau dokumen lama dengan nama yang berbeda.Open : untuk membuka kembali dokumen yang telah disimpanClose : untuk menutup lembar kerja

2. Tab menu home

Tab ini paling sering digunakan, karena menggabungkan semua fitur format teks seperti font, dan pengaturan paragraf. Dalam menu home berisi ribbon untuk pengaturan standart, antara lain :

Ribbon Clipboard : berisi ikon untuk mengedit teks.Ribbon Font : untuk mengatur hurufRibbon Paragraph : untuk mengatur paragraph teksRibbon Styles : untuk mengatur model penulisanRibbon Editing : untuk mencari karakter, kalimat pada dokumen atau menyeleksi teks pada dokumen

3. Tab menu Insert

Tab ini digunakan untuk memasukkan berbagai item dalam dokumen seperti gambar, clip art, halaman cover, tabel, text box, penomeran halaman, dan lain-lain.

4. Tab menu page layout

Tab ini digunakan untuk mengatur halaman seperti margin, tema, orientasi, dan lain-lain.

5. Tab menu References

Tab ini menyediakan banyak solusi sederhana untuk menghasilkan dokumen

6. Tab menu Mailings

Tab ini digunakan untuk mengirim surat seperti mencetak amplop dan label.

Ribbon CreateRibbon Start Mail MergeRibbon Write & Insert FieldsRibbon Preview ResultsRibbon Finish

7. Tab menu Review

Tab ini untuk membuat perubahan dalam dokumen yang dimungkinkan karena masalah ejaan atau tata bahasanya.

Ribbon ProofingRibbon LanguageRibbon CommentsRibbon TrackingRibbon ChangesRibbon CompareRibbon Protect

8. Tab menu View

Tab ini digunakan untuk mengubah tampilan dokumen kita ke dua halaman yang berbeda dan memperbesar/memperkecil halaman.

Ribbon Document ViewsRibbon ShowRibbon ZoomRibbon WindowsRibbon Macros

menu yang berisi beberapa perintah standar di dalam Microsoft Office Word. 32. Jelaskan pengertian Tab pada microsoft excel dan sebutkan contohnya


Penjelasan:

Tab data dan tab formula ialah contoh tab ribbon yang dapat diakses dalam Ms. office excel versi 2007 dan versi berikutnya.

Fungsi tab data pada Ms. office Excel: menampung group-group ikon, seperti group Get & Transform, Connection, Sort & Filter, Data Tools, Forecast, Outline.

Fungsi tab formulas pada Ms. office Excel: menampung group-group ikon, seperti group Function Library, Defined names, Formula auditing, Calculation.

Jawaban:

Misalkan pada Microsoft excel versi lama:

tab yaitu bagian yg muncul pada saat tampil kotak dialog Format Cells.

Misalkan pada Microsoft excel versi baru:

tab yaitu bagian tampilan Ms. Excel yg berfungsi menampung group-group tertentu.

Sedangkan group-group di dalam tab, berfungsi menampung icon-icon khusus sesuai fungsinya.

Penjelasan:

Yang termasuk Ms. Excel versi lama adalah Ms. Excel 2003, Ms. Excel 2000, Ms. Excel 97

Yang termasuk Ms. Excel versi baru adalah Ms. Excel 2007, Ms. Excel 2010, Ms. Excel 2013

Perhatikan gambar berikut ini:

- tanda panah merah menunjukkan cara menampilkan Format Cells

- tanda panah hijau menunjukkan tab-tab yg ada di Formt Cells

- tanda panah biru menunjukkan tab-tab yg ada dalam Ms. Excel versi baru. (antara lain: Home, Insert, Page Layout, Formulas)
