19. Dalam satu paket soal ujian terdiri dari paket A sebanyak 5 soal dan paket B sebanyak 6 soal.banyak cara menjawab 5 soal,jika 3 soal dari paket A dan 2 soal dari paket B adalah...cara


Menggunakan kombinasi berdasarkan variasi soal adalah sama apabila berbeda urutan.
Berlaku:
n = Soal A x Soal B
n = C(5,3) x C(6,2)
n = 5!/(2!3!) x 6!/(4!2!)
n = 5.4/2 x 6.5/2
n = 10 x 15
n = 150 cara

38. soal ujian terdiri 6 soal paket A dan 4 soal paket B setiap siswa wajib mengerjakan 8 soal yg terdiri dari 5 soal paket A dan 3 soal paket B. jika soal no 1 dan 2 dipaket A wajib dikerjakan, tentukan banyaknya cara siswa mengerjakan soal ujian​


5 soal dari paket A dan nomor 1 adalah wajib. Jadi, siswa hanya perlu memilih 4 soal lagi dari paket A.

3 soal dari paket B dan nomor 1 adalah wajib. Jadi, siswa hanya perlu memilih 2 soal lagi dari paket B.

Sehingga, banyak cara mengerjakan soal yg dapat dilakukan adalah :

5C4 x 3C2 = [5!/(4!1!)] x [3!/(2!1!)]

= 5 x 3

= 15
