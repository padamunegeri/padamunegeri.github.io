22. pilihlah jawaban yang tepat​


Jawaban:

A. {-p - 19} / 6p

Penjelasan dengan langkah-langkah:

aljabar..

= {p - 2} /3p - {p + 5} / 2p

= 2p{p - 2} - 3p{p + 5} / {6p²}

= {2p² - 4p - 3p² - 15p} / 6p²

= {-p² - 19p} / 6p²

= p {-p - 19} / p {6p}

= { -p - 19 } / 6p
