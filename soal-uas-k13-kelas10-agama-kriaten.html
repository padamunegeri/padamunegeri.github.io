2. ada yang bisa jelasin cara jawabnya buat belajar uas besok kelas10


selesai
happyMathday........[tex] \frac{1 - 6x}{5 - x} \geqslant 7 \\ \\ \frac{1 - 6x}{5 - x} - 7 \geqslant 0 \\ \\ \frac{1 - 6x}{5 - x} - \frac{35 - 7x}{5 - x} \geqslant 0 \\ \\ \frac{ x - 34}{5 - x} \geqslant 0[/tex]
Masuk ke pembuat nol :

x - 34 = 0 → x = 34

5 - x = 0 → x = 5

cek x = 0

0-34 / 5-0 = -34/5 (hasilnya negatif)

cej x = 6

6-34 / 5-6 = -28/-1 (hasilnya positif)

maka 5 < x ≤ 34 memiliki nilai positif. karena bertanda "≥" maka nilai x mengikuti hasil positif .nilai x berada di 5 < x ≤ 34

HP = {x | 5 < x ≤ 34, x € R}

Note: Jangan menuliskan 5 ≤ x ≤ 34 karena jika x = 5 maka penyebutnya akan bernilai 0

8. 2 Soal Kak#Matematika#Kelas10​


Nomor 1b

[tex]x_1<x_2=(x_1+2x_2)^2-2x_2[/tex]

pandang

[tex]x_2=(x_1+2x_2)^2-2x_2=(x_1^2+4x_1x_2+4x_2^2)-2x_2[/tex]

[tex]0=(x_1^2+4x_1x_2+4x_2^2)-3x_2[/tex]

[tex]0=4[x_2^2]+(4x_1-3)[x_2]+x_1^2[/tex]

dengan rumus abc, dapat dicari solusi [tex]x_2[/tex]

[tex]x_2=\frac{-(4x_1-3) \pm\sqrt{(4x_1-3)^2-4(4)(x_1^2)} }{2(4)}[/tex]

[tex]=\frac{1}{8} (-4x_1+3 \pm\sqrt{16x_1^2-24x_1+9-16x_1^2} )[/tex]

[tex]=\frac{1}{8} (-4x_1+3 \pm \sqrt{9-24x_1} )[/tex]

[tex]=-\frac{1}{2} x_1+\frac{3}{8} \pm\sqrt{9-24x_1}[/tex]          dimana [tex]x_1 \leq \frac{3}{8}[/tex]

sehingga

[tex]x_1<-\frac{1}{2} x_1+\frac{3}{8} \pm\sqrt{9-24x_1}[/tex]

[tex]\frac{3}{2}x_1-\frac{3}{8} < \pm\sqrt{9-24x_1}[/tex]

karena

[tex]x_1\leq \frac{3}{8} \\\frac{3}{2} x_1-\frac{3}{8} \leq \frac{3}{2} \cdot \frac{3}{8} -\frac{3}{8}=\frac{3}{16}[/tex]

bagi dua kasus

kasus 1:

untuk     [tex]0\leq \frac{3}{2} x_1-\frac{3}{8} \leq \frac{3}{16}[/tex]

[tex]\frac{3}{2}x_1-\frac{3}{8} <\sqrt{9-24x_1}[/tex]

diperoleh

[tex]\frac{1}{4} \leq x_1< \frac{2}{3} \sqrt{67} -\frac{61}{12}[/tex]

[tex]0.25\leq x_1<0.37356851458163331330[/tex]

kasus 2:

untuk     [tex]\frac{3}{2} x_1-\frac{3}{8} <0[/tex]

[tex]\frac{3}{2}x_1-\frac{3}{8} < - \sqrt{9-24x_1}[/tex]

diperoleh

[tex]x_1<-\frac{2}{3} \sqrt{67} -\frac{61}{12} \approx -10.540235181248299980[/tex]

Nomor 2b

[tex](\frac{1}{3} )^{x^2-6x-16}=(\frac{1}{24})^{x-2}\\(3)^{x^2-6x-16}=(24)^{x-2}[/tex]

karena basisnya berbeda [tex]3[/tex] dan [tex]24[/tex],

dan [tex]a^0=1[/tex] untuk semua bilangan real [tex]a\neq 0[/tex]

[tex]x^2-7x-14=0[/tex]  dan  [tex]-3(x-2)=0[/tex]

tidak ada solusi dari dua persamaan di atas.

CARI LAGI!

[tex](3)^{x^2-6x-16}=(24)^{x-2}\\(3)^{x^2-6x-16}=(3)^{x-2} \cdot (8)^{x-2}\\(3)^{x^2-6x-16-(x-2)}=(8)^{x-2}\\[/tex]

perlu disamakan basisnya

karena [tex]8=3^{\log_3 8}[/tex] maka

[tex](3)^{x^2-7x-14}=(3^{\log_3 8})^{x-2}\\(3)^{x^2-7x-14}=(3)^{(x-2)\cdot \log_3 8}[/tex]

diperoleh

[tex]x^2-7x-14=(x-2)\cdot \log_3 8[/tex]

1). |8x + 4| = 6

|8x + 4| = {8x + 4, jika 8x + 4 >= 0} {-(8x + 4), jika 8x + 4 < 0} 8x < -4 x < -2

|8x + 4| = 6 atau -(8x + 4) = 68x + 4 = 6 -8x - 4 = 68x = 6 - 4 -8x = 6 + 48x = 2 -x = 10/8 (dikali -1)x = 8/2 x = -8/10x = 4

2). |3x + 7| = 3

|3x + 7| = {3x + 7, jika 3x + 7 >= 0} {-(3x + 7), jika 3x + 7 < 0} 3x < -7 x < -7/3

|3x + 7| = 3 atau -(3x + 7) = 33x + 7 = 3 -3x - 7 = 33x = 3 - 7 -3x = 3 + 73x = -4 -x = 3/10 (dikali -1)x = -3/4 x = -3/10

3). |6x - 4| = 7

|6x - 7| = {6x - 4, jika 6x - 4 >= 0} {-(6x - 4), jika 6x - 4 < 0} 6x < 4 x < 4/6

|6x - 4| = 7 atau -(6x - 4) = 73x - 4 = 7 6x + 4 = 73x = 7 + 4 6x = 7 - 43x = 11 x = 6/3x = 3/11 x = 2
