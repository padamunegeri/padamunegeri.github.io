Domain Name System atau DNS memiliki peranna yang cukup penting dalam penggunaan internet. DNS memiliki tugas untuk menerjemahkan nama suatu host menjadi ip address ataupun sebaliknya dari ip address menjadi nama sebuah host.

Sebagai contoh paling mudah, ketika kita melakukan ping ke nama host www.google.com dapat terlihat reply di dapatkan dari alamat 216.239.38.120, alamat ip inilah yang mewakili suatu host di dalam sebuah jaringan. Jadi kurang lebih cara kerja DNS bisa dianalogikan seperti buku telepon dalam kehidupan sehari - hari, daripada kita mengingat kumpulan angka - angka kita akan lebih mudah mengingat nama yang kita berikan, di sinilah peran DNS untuk menerjemahkan nama host menjadi ip address.

38. penggunaan sistem DNS(DOMAIN NAME SERVICE) pada internet mulai dikembangkan pada?


Paul Mockapetris menemukan DNS pada tahun 1983 ;
spesifikasi asli muncul di RFC 882 dan 883. Tahun 1987,
penerbitan RFC 1034 dan RFC 1035 membuat update
terhadap spesifikasi DNS. Hal ini membuat RFC 882 dan
RFC 883 tidak berlaku lagi. Beberapa RFC terkini telah
memproposikan beberapa tambahan dari protokol inti DNS.

39. Apa pengertian dari domain , kodomain , range dan fungsi dalam hal matematika?


domain itu daerah asal
kodomain itu daerah kawan
range itu daerah hasil

40. Apa yang dimaksud dengan Domain Name?


Domain name itu nama yang diberikan untuk mengidentifikasikan server yang diakses di jaringan komputer maupun di Internet.
