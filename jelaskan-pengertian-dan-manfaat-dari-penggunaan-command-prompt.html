Command prompt merupakan salah satu fitur dari microsoft windwos yang menggunakan tampilan teks, command promt sendiri memiliki fungsi untuk mengecek hardisk, melakukan copy paste dan perintah lainnya. Untuk mengakses command prompt kita dapat mengaksesnya melalui start menu atau melalui run.

Beberapa manfaat dari command prompt yang berkaitan dengan virus adalah sebagai berikut

1. Menampilan file-file yang tersembunyi oleh virus, kita dapat menggunakan perintah atrrib pada command prompt.

2. Membuka file yang tersembunyi oleh virus, beberapa virus melakukan penyembunyian terhadap file asli kita, dengan command prompt kita bisa mengakses file tersebur, caranya mengarahkan command prompt pada lokasi file tersebut dan mengetik nama file tersebut.

3. Menghapus virus, dengan mengetahui nama dan ekstensi dari sebuah virus, kita dapat melakukan penghapusan secara massal tanpa harus mengakses file virus tersebut.

36. apa bedanya command line sama command prompt??


Command line merupakan tempat penulisan
command yang terdapat pada Linux.

1. Direktorinya dalam bentuk file
2. Prompt ada 2 yaitu dapat berfungsi sebagai
user ($) atau juga sebagai root (#).
3. Banyak sesi yang digunakan untuk
menyelesaikan satu perintah.
4. Mode yang digunakan ada dua,yaitu mode
grafis dengan mode non grafis (tty)
Command Prompt Windows
Command Prompt merupakan tempat penulisan
command yang terdapat pada Windows.
Perbedaannya juga dapat dilihat di bawah :
1. Direktorinya dalam bentuk drive
2. Prompt nya hanya drive
3. Hanya satu sesi yang digunakan untuk
menyelesaikan satu perintah
4. Mode yang digunakan mode grafis ( desktop)
