Open source Software

merupakan software yang dapat dikembangkan sendiri oleh pengguna seperti source code dan interface software karena software yang bersifat opensource sendiri tidak menggunakan hak cipta dalam pembuatannya. Open source software tidak perlu mengeluarkan biaya untuk membelinya.

Licensed Software

merupakan software yang memiliki hak cipta dan cenderung berbayar untuk mendapatkan akses lengkap pada fiturnya. Source code dan modifikasi tidak dapat dilakukan pada software jenis ini karena adanya hak cipta pula. Software ini juga lebih difokuskan untuk komersial daripada pengembangan bebas.
