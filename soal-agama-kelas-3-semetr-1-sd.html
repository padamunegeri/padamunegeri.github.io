y = 3x² - 7x
maka:
a = 3
b = -7
c = 0
sumbu simetri
[tex]x = \frac{ - b}{2a} \\ x = \frac{ - ( - 7)}{2(3)} \\ = \frac{7}{6} = 1.16[/tex]
a = 3 grafik fungsi terbuka ke atas maka memiliki nilai minimum.
[tex]nilai \: minimum \: fungsi \: -> f = f( \frac{7}{6} ) \\ y = 3 {x}^{2} - 7x \\ f( \frac{7}{6} ) = 3 {x}^{2} - 7x \\ = 3( \frac{7}{6} ) {}^{2} - 7( \frac{7}{6} ) \\ = 3( \frac{49}{36} ) - \frac{49}{6} \\ = \frac{147}{ 36} - \frac{49}{6} \\ = \frac{147 - 294}{36} \\ = \frac{ - 147}{36} \\ = - 4.08[/tex]

17. bagaimana cara menetukan semetri lipat dan semetri putar pada bangun datar?


Dengan cara melipat atah memutar pada bangun datarsimetri lipat = dengan melipat suatu bangun datar sehingga ada satu garis diagonal yg membagi 2 daerah sama besar.
simetri putar = dengan memutar suatu bangun datar tersebut, lalu menentukan berapa kali bangun datar tersebut menempati posisi asalnya dalam satu kali putaran.
