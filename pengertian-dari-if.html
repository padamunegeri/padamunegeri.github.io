Jawaban:

Penjelasan:

1. If Clause + Reminder

Merupakan bentuk kalimat yang digunakan untuk mengingatkan konsekuensi yang harus dilakukan jika menginginkan sesuatu.

IF + SIMPLE PRESENT, S + HAVE TO/HAS TO/ MUST + V1

Contoh:

Dad: Do you want to join the next mathematics competition?

Tom: Yes, I do dad, and I really want to be the winner.

Dad: If you want to be the winner, you have to study hard.

Tom: Thanks, dad.

2. If Clause + Suggestion

If clause ini digunakan didalam kalimat untuk memberikan suatu saran mengenai suatu tindakan yang harus dilakukan jika menginginkan suatu kejadian terjadi. Bentuk saran dan bentuk peringatan berbeda, saran bisa dilakukan atau tidak (tergantung level), sedangkan peringatan harus dilakukan karena hal itu merupakan suatu konsekuensi.

IF + SIMPLE PRESENT, S + SHOULD /HAD BETTER/ OUGH TO + V1

Contoh:

Dad: Why do you look so pale?

Tom: Hmm I’m sick dad. I may get a fever.

Dad: Ok, If you want to get better soon, you should get some medicine and take a rest.

Tom: Thanks, dad.

IF + SIMPLE PRESENT, WHY DON’T + S + V1?

Jennie: I really like watching, but sometimes I don’t know the latest movie. Do you have any idea?

Anna: If you want to know the latest movie, why don’t you search for it on the internet?

Jennie: That’s a good idea. Thanks

3. If Clause + Imperative

Digunakan untuk menyuruh seseorang untuk melakukan sesuatu jika meginginkan sebuah kondisi terjadi.

IF + SIMPLE PRESENT, V1 + O

Contoh: If you want to know her more, follow her on Instagram.

﻿

4. If Clause + General Truth

Kalimat ini digunakan untuk menyatakan fakta yang terjadi atau akan terjadi jika menginginkan sebuah kondisi terjadi.

IF + SIMPLE PRESENT, SIMPLE FUTURE/PRESENT

Contoh: If you are 17 years old or older, you are allowed to have an id card.

﻿

5. If Clause + Dream

Digunakan untuk menyatakan sebuah mimpi di masa depan tentang apa saja yang mungkin terjadi.

IF + SIMPLE PRESENT, SIMPLE FUTURE

Contoh: If I become a doctor, I will be a good doctor.

If Clause Type 1 : 
-Kondisi yang memungkinkan untuk dicapai di masa mendatang
Ex: If I study hard, I will pass the exam ( Jika saya belajar dengan keras, saya akan lulus ujian )
If clause : Simple present ( Verb 1 )
Main clause : Future tense ( Will )
Ex: I will achive my dream if I study hard 

If Clause Type 2 :
-Kondisi yang memungkinkan untuk dicapai dengan memprediksi 
Ex: If I studied, I would pass the exam ( Seandainnya saya belajar, saya pasti lulus ujian )
If clause : Simple past ( Verb 2 ) 
Main clause : Would + Invinitive
Ex: I would pass the exam if I studied hard

If Clause Type 3 :
-Kondisi yang tidak mungkin untuk dicapai karena sudah telat
Ex: If i had studied, I would have passed the exam ( Jika waktu itu saya belajar, saya sudah lulus ujian )
If clause : Past perfect
Main clause : Would + have + past participle 
Ex: I would have passed the exam, if I had studied

22. 1. Carilah Pengertian, Fungsi, Penulisan (Syntax), Contoh penggunaan di dunia nyata, dari rumus di bawah ini : a. SUMIF b. SUMIFS c. COUNTIF d. COUNTIFS e. NESTEDIF f. IF ERROR ​


Jawaban:

a. SUMIF

Pengertian: SUMIF adalah sebuah rumus di Excel yang digunakan untuk menjumlahkan sel-sel tertentu pada rentang sel tertentu yang memenuhi kriteria tertentu.Fungsi: SUMIF digunakan untuk menghitung jumlah nilai pada rentang sel tertentu yang memenuhi kriteria tertentu.Penulisan (Syntax): =SUMIF(range, criteria, [sum_range])Contoh penggunaan di dunia nyata: Misalnya, Anda ingin menghitung total penjualan produk A selama bulan Januari. Anda dapat menggunakan rumus =SUMIF(A2:A10,"Produk A",B2:B10) untuk menjumlahkan nilai pada sel B2:B10 di mana sel di sampingnya pada kolom A memenuhi kriteria "Produk A".

b. SUMIFS

Pengertian: SUMIFS adalah sebuah rumus di Excel yang digunakan untuk menjumlahkan sel-sel tertentu pada rentang sel tertentu yang memenuhi beberapa kriteria tertentu.Fungsi: SUMIFS digunakan untuk menghitung jumlah nilai pada rentang sel tertentu yang memenuhi beberapa kriteria tertentu.Penulisan (Syntax): =SUMIFS(sum_range, criteria_range1, criteria1, [criteria_range2, criteria2], ...)Contoh penggunaan di dunia nyata: Misalnya, Anda ingin menghitung total penjualan produk A selama bulan Januari dan Februari. Anda dapat menggunakan rumus =SUMIFS(B2:B10, A2:A10, "Produk A", C2:C10, ">1/1/2022", C2:C10, "<2/28/2022") untuk menjumlahkan nilai pada sel B2:B10 di mana sel di sampingnya pada kolom A memenuhi kriteria "Produk A" dan pada kolom C memenuhi kriteria rentang tanggal tertentu.

c. COUNTIF

Pengertian: COUNTIF adalah sebuah rumus di Excel yang digunakan untuk menghitung jumlah sel pada rentang sel tertentu yang memenuhi kriteria tertentu.Fungsi: COUNTIF digunakan untuk menghitung jumlah sel pada rentang sel tertentu yang memenuhi kriteria tertentu.Penulisan (Syntax): =COUNTIF(range, criteria)Contoh penggunaan di dunia nyata: Misalnya, Anda ingin menghitung berapa banyak mahasiswa yang mendapat nilai A pada ujian. Anda dapat menggunakan rumus =COUNTIF(B2:B10,"A") untuk menghitung jumlah sel pada sel B2:B10 yang bernilai "A".

e. NESTEDIF

Pengertian: NESTEDIF adalah sebuah fungsi dalam Excel yang digunakan untuk mengevaluasi beberapa kondisi atau kriteria yang tertanam dalam satu sama lain.Fungsi: NESTEDIF digunakan untuk mengembalikan nilai berdasarkan beberapa kondisi atau kriteria tertentu yang tertanam dalam satu sama lain.Penulisan (Syntax): =IF(kondisi1, nilai1, IF(kondisi2, nilai2, IF(kondisi3, nilai3, ...)))Contoh penggunaan di dunia nyata: Misalnya, Anda ingin menentukan apakah sebuah karyawan berhak mendapatkan bonus berdasarkan penjualan mereka. Jika penjualan karyawan di atas $100.000, maka mereka berhak atas bonus sebesar 10% dari total penjualan. 39. pengertian, jenis, dan ciri If clause​


If clause adalah bentuk kalimat pengandaian dengan menggunakan “if”. Ada empat tipe if clause, pertama yaitu tipe 0 dengan tenses (simple present + simple present). Tipe 0 digunakan apabila mengungkapkan pengandaian yg bersifat fakta (general truth) e.g. : if you drop an egg, it cracks. Tipe 1 digunakan apabila mengungkapkan kemungkinan (possibility) dengan tenses (simple present + future tense) e.g. If the teacher comes, he will congratulate us for our win. Tipe 2 digunakan apabila kita mengandaikan sesuatu yg mustahil terjadi (fantasy) tenses yg digunakan adalah (simple past + past future) e.g. : if i were you, I would tell her that I loved her (faktanya: aku bukan kamu). Tipe 3 digunakan apabila kita mengungkapkan sesuatu yg diandaikan namun hasilnya tidak sesuai atau biasa disebut tipe penyesalan (regret) tenses yg digunakan adalah (past perfect + past future perfect) e.g. : if I had studied last night, I would have passed the exam (faktanya: saya tidak belajar dan saya tidak lulus ujian)
