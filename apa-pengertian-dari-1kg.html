z = 1 kg jeruk = Rp. 21.000

Penjelasan dengan langkah-langkah:

ditentukan :

1 kg alpukat = x

1 kg jambu = y

1 kg jeruk = z

ditanya, 1 kg jeruk adalah...

persamaan :

x + y + z = 48.000

1) 2x + y + z = 60.000

x + y + z = 48.000

--------------------------------- -

x = 12.000

2) x + 2y + z = 63.000

x + y + z = 48.000

--------------------------------- -

y = 15.000

x + y + z = 48.000

z = 48.000 - x - y

= 48.000 - 12.000 - 15.000

= 21.000

14. Diketahui harga 4 kg salak,1kg jambu,2kg klengkeng adalah rp54.000.00.harga 1kg slak,2kg jambu,2kg klengkeng adalah rp43.000.00.harga 3kg slak,1kg jambu, 1kg klengkeng adalah rp37.750.00.berpa harga 1kg slak,1kg jambu 1kg klengkeng


Jawab:

1 kg salak = Rp 6.5001 kg jambu = Rp 8.5001 kg kelengkeng = Rp 9.750

Penjelasan dengan langkah-langkah:

Misal:

salak = x

jambu = y

klengkeng = z

4x + y + 2z = 54.000 .... (1)

x + 2y + 2z = 43.000 ..... (2)

3x + y + z = 37.750    ..... (3)

Eliminasi (1) dan (3)

4x + y + 2z = 54.000

3x + y + z = 37.750

------------------------------   -

x + z = 16.250  ...... (4)

Eliminasi (2) dan (3)

x + 2y + 2z = 43.000 x 1

3x + y + z = 37.750 x 2

sehingga

x + 2y + 2z = 43.000

6x + 2y + 2z = 75.500

--------------------------------  -

-5x = - 32.500

x = 6.500 <== harga salak 1 kg

Substitusi pers. (4)

x + z = 16.250

6.500 + z = 16.250

z = 9.750 <== harga kelengkeng 1 kg

Substitusi pers (3)

3x + y + z = 37.750

3(6.500) + y + 9.750 = 37.750

y = 37.750 - 29.250

y = 8.500 <== harga jambu 1 kg

Jadi kesimpulannya:

1 kg salak = Rp 6.5001 kg jambu = Rp 8.5001 kg kelengkeng = Rp 9.750

21. diketahui harga 4kg salak, 1kg jambu, dan 2kg kelengkeng adalah rp.100.000,00. harga 3kg salak, 1kg jambu, dan 1kg Kelengkeng adalah 75.000,00. tentukan harga 1kg salak, 1kg jambu, 1kg kelengkeng​


Jawab:

Dalam menentukan harga 1kg masing-masing buah, kita bisa menggunakan metode regresi dari data harga yang diketahui. Misalnya, menentukan harga 1kg salak dengan persamaan regresi sebagai berikut:

harga = a * kg salak + b * kg jambu + c * kg kelengkeng

Ketahui harga dan banyaknya kg untuk 2 data harga yang diketahui (4kg salak, 1kg jambu, 2kg kelengkeng dan 3kg salak, 1kg jambu, 1kg kelengkeng).

4kg salak, 1kg jambu, 2kg kelengkeng:

harga = 100.000

kg salak = 4

kg jambu = 1

kg kelengkeng = 2

3kg salak, 1kg jambu, 1kg kelengkeng:

harga = 75.000

kg salak = 3

kg jambu = 1

kg kelengkeng = 1

Masukkan data tersebut ke dalam matriks X dan Y:

X = [[4, 1, 2], [3, 1, 1]]

Y = [[100.000], [75.000]]

Hitung koefisien regresi a, b, dan c dengan memecahkan persamaan regresi dengan metode matriks.

Hasil dari koefisien regresi a, b, dan c adalah sebagai berikut:

a = 22.500

b = 25.000

c = 25.000

Dengan hasil koefisien regresi tersebut, maka harga 1kg salak, 1kg jambu, dan 1kg kelengkeng adalah:

harga 1kg salak = 22.500

harga 1kg jambu = 25.000

harga 1kg kelengkeng = 25.000
