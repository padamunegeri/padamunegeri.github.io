Jika Chandra ingin membeli 5 buku,10 pensil, dan 6 pulpen maka Chandra harus menyediakan sejumlah uang..........................


misal :
buku = x
pensil = y
pulpen = z

2x + 3y + z = 17.000 ....(1)
2x + 2y + 2z = 20.000 ....(2)
3x + 4y + 3z = 32.000 ....(3)

eliminasikan (1) dan (2)
2x + 3y + z = 17.000
2x + 2y + 2z = 20.000
__________________ -
y - z = -3.000 ....(4)

eliminasikan (2) dan (3)
2x + 2y + 2z = 20.000 (× 3)
3x + 4y + 3z = 32.000 (× 2)

6x + 6y + 6z = 60.000
6x + 8y + 6z = 64.000
_________________ -
-2y = -4.000
y = -4.000/-2 = Rp 2.000,00

masukkan nilai y ke (4)
y - z = -3.000
2.000 - z = -3.000
-z = -3.000 - 2.000
-z = -5.000
z = Rp 5.000,00

masukkan nilai y dan z ke (1)
2x + 3y + z = 17.000
2x + 3(2.000) + 5.000 = 17.000
2x + 6.000 + 5.000 = 17.000
2x + 11.000 = 17.000
2x = 17.000 - 11.000
2x = 6.000
x = 6.000/2 = Rp 3.000,00

5x + 10y + 6z
= 5(3.000) + 10(2.000) + 6(5.000)
= 15.000 + 20.000 + 30.000
= Rp 65.000,00

Jadi, Chandra harus menyediakan sejumlah Rp 65.000,00 untuk membeli 5 buku, 10 pensil, dan 6 pulpen

Psikotes memiliki arti sebagai bentuk evaluasi atau tes tertulis, visual atau verbal yang dilakukan oleh psikolog untuk menilai fungsi kognitif dan emosional seseorang. Pada umumnya, tes psikotes dilakukan oleh perusahaan untuk mengetahui karakter dan kepribadian kamu.

[tex] \frac{besarnya \: diskon}{harga \: awal} \times 100[/tex]
